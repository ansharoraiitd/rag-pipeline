# RAG Pipeline

A Retrieval Augmented Generation system — chat with your own documents.

## What this will be
A complete RAG pipeline where users can upload any document
and ask questions about it. The system retrieves relevant chunks
and generates accurate, grounded answers.

## Progress
- [x] Day 1: Embeddings — semantic search foundation
- [x] Day 2: ChromaDB — storing and searching embeddings
- [ ] Day 3: Chunking — splitting documents intelligently
- [ ] Day 4: Full RAG chain — retrieval + generation
- [ ] Day 5: Chat with documents — complete app

## Tech stack
Python · LangChain · ChromaDB · Gemini API · Embeddings

## Setup
pip install -r requirements.txt
Add GEMINI_API_KEY to .env
